1. Visão Geral

O RCO Manager é um software de automação para professores que utilizam o sistema RCO (Registro de Classe Online). Ele conecta seu diário de classe no Google Sheets com o RCO, automatizando tarefas como lançamento de notas, sincronização de alunos e registro de comentários/ocorrências.

O que o software fazComo faz
Lança notas no RCOLê as notas da planilha e preenche o RCO automaticamente via Chrome
Sincroniza alunosCompara a lista de alunos do RCO com a planilha e adiciona/remove automaticamente
Registra comentários/ocorrênciasLê ocorrências da planilha e lança no RCO com texto formatado
Gera diários de classeCria planilhas Google Sheets estruturadas no seu Google Drive
Atualiza resumo de notasLê notas finais do RCO e preenche aba de Resumo na planilha
⚠
O software controla o Chrome automaticamente. Durante operações de lançamento, não clique no navegador Chrome — aguarde o processo terminar.

2. Configuração Inicial

Antes de usar qualquer funcionalidade, é necessário realizar três configurações. Elas aparecem na barra superior do software.

2.1 Autorizar Google Sheets

Esta autorização permite que o software leia e escreva nas suas planilhas Google Sheets e crie arquivos no seu Google Drive.

  1. Clique no botão Autorizar Google na barra superior.
  2. Uma janela do navegador será aberta com a tela de login do Google.
  3. Faça login com a conta Google que possui acesso às planilhas.
  4. Autorize o acesso solicitado.
  5. A barra superior mudará para ✓ Google autorizado.
ℹ
Esta autorização é salva localmente. Você só precisará repetir este processo se trocar de computador ou revogar o acesso manualmente.

2.2 Conectar Chrome ao RCO

O software utiliza o Chrome instalado no computador para acessar o RCO. É necessário estar logado no RCO dentro do Chrome.

  1. Clique em Conectar Chrome na barra superior.
  2. O Chrome abrirá automaticamente na página de login do RCO.
  3. Faça login no RCO normalmente dentro do Chrome.
  4. Após entrar no sistema, volte ao RCO Manager e clique em Confirmar login.
  5. O software verificará a conexão. Se bem-sucedido, exibirá ✓ Chrome conectado.
⚠
Se fechar o Chrome ou a sessão do RCO expirar, será necessário clicar em Reconectar e repetir o processo de login.

2.3 Configurar Pasta do Google Drive

Defina em qual pasta do Google Drive as planilhas geradas pelo software serão salvas.

  1. Acesse a aba Configurações no menu lateral.
  2. Abra o Google Drive no navegador e navegue até a pasta desejada.
  3. Copie o ID da pasta na URL — é o trecho após /folders/.
    Exemplo: https://drive.google.com/drive/folders/1MsRODhlM...
  4. Cole o link completo ou apenas o ID no campo indicado.
  5. Clique em Salvar.
ℹ
A pasta deve ser acessível pela conta Google autorizada no passo anterior. Você pode colar o link completo — o software extrai o ID automaticamente.

3. Dashboard

O Dashboard é a tela inicial do software. Nele você tem uma visão geral das suas turmas e pode acessar as principais funcionalidades.

Cartões de resumo

CartãoO que mostra
TurmasTotal de turmas cadastradas
AlunosTotal de alunos ativos (inativos não contabilizados)
Última atualizaçãoData e hora da última sincronização com o RCO

Atualizar Banco de Dados

Esta função lê as turmas, alunos e avaliações diretamente do RCO e salva localmente. É o ponto de partida — execute sempre que iniciar o software ou quando houver mudanças no RCO.

  1. Clique em Atualizar Banco no Dashboard.
  2. Selecione o trimestre desejado.
  3. Marque as turmas que deseja sincronizar.
  4. Clique em Iniciar e aguarde o progresso.

Lista de turmas

Cada turma exibe nome, escola, disciplina e quantidade de alunos. Clique sobre uma turma para expandir e ver as avaliações lançadas por trimestre, com data de lançamento e status.

4. Planilhas

Nesta aba você gerencia o vínculo entre cada turma e sua planilha Google Sheets. Uma turma pode estar em três estados:

StatusSignificado
Sem planilhaNenhuma planilha vinculada. Não é possível lançar notas.
VinculadaPlanilha associada. Todas as funcionalidades disponíveis.

4.1 Gerar nova planilha

Cria um diário de classe do zero no Google Drive, já estruturado para uso com o RCO Manager.

  1. Clique em Gerar na turma desejada.
  2. Selecione o Modo:
    • Diário — notas calculadas a partir de penalidades por aula (faltas, celular, etc).
    • Completo — igual ao Diário, com mais recursos de controle.
    • Simples — notas inseridas diretamente pelo professor, sem penalidades.
  3. Selecione o Trimestre (1º, 2º ou 3º).
  4. Defina a frequência semanal de aulas (quantas vezes por semana).
  5. Configure as avaliações (nome, valor máximo e pesos).
  6. Clique em Gerar e aguarde a criação no Google Drive.
✓
Após gerar, a planilha já aparece vinculada à turma automaticamente. Você pode clicar em Abrir para visualizá-la no Google Sheets.

4.2 Usar planilha existente

Vincula um diário de classe já existente (criado por você) ao invés de gerar um novo. O software utilizará as colunas dessa planilha para todas as operações.

  1. Clique em Usar existente na turma desejada.
  2. Cole o link completo da planilha ou apenas o ID.
  3. Clique em Vincular.
⚠
A planilha externa precisa seguir uma estrutura mínima para funcionar corretamente. Consulte a seção 8. Estrutura esperada de planilha externa.

4.3 Diagnóstico de compatibilidade

Para turmas vinculadas, é possível verificar se a planilha possui todas as colunas e abas reconhecidas pelo software.

  1. Clique sobre a linha da turma vinculada (onde está escrito ▼ Ver compatibilidade).
  2. O painel expandirá e iniciará a análise automaticamente.
  3. Cada módulo será exibido com ✓ (encontrado) ou — (não encontrado).

Módulos verificados:

MóduloO que verifica
Alunos (Nome + Nº)Colunas de nome e número de chamada
SituaçãoColuna de status do aluno (ativo/inativo)
AvaliaçõesColunas de notas (ATV 1, ATV 2, REC 1…)
Datas de aulaLinha com datas no formato DD/MM
TrimestresAbas nomeadas 1º Tri, 2º Tri, 3º Tri
OcorrênciasColunas de lançamento de comentários

4.4 Gerar em Lote

Cria planilhas para múltiplas turmas de uma só vez, com as mesmas configurações.

  1. Clique em Gerar em Lote no topo da aba.
  2. Selecione as turmas (use Sem Planilha para filtrar apenas as que precisam).
  3. Configure Modo, Trimestre, Frequência e Avaliações.
  4. Clique em Gerar e acompanhe o progresso.

4.5 Resumo em Lote

Atualiza a aba "Resumo" de múltiplas planilhas com as notas finais lidas diretamente do RCO.

  1. Clique em Resumo em Lote no topo da aba.
  2. Selecione o trimestre.
  3. Marque as turmas desejadas.
  4. Clique em Gerar Resumo.
ℹ
O resumo mostra aprovados (verde), reprovados (vermelho) e situação parcial (amarelo) para cada aluno, com base nas notas finais do RCO.

5. Alunos

Esta aba compara a lista de alunos do RCO com a lista na planilha e mantém as duas sincronizadas.

✓ SincronizadaRCO e planilha estão iguais

5.1 Verificar e Sincronizar

Para uma turma individual:

  1. Clique em Verificar na turma desejada.
  2. O software compara os alunos do RCO com a planilha e exibe as diferenças:
    • + Aluno novo no RCO, ainda não na planilha
    • ~ Situação do aluno mudou (ex: ativo → transferido)
    • − Aluno na planilha mas não encontrado no RCO
  3. Clique em Sincronizar para aplicar as alterações na planilha.

5.2 Verificar Todas e Sincronizar em Lote

  1. Clique em Verificar Todas para comparar todas as turmas de uma vez.
  2. Acompanhe a barra de progresso.
  3. Após verificar, clique em Sincronizar em Lote para corrigir todas as pendências.
  4. Selecione as turmas e confirme.

6. Notas

Esta aba permite lançar ou editar notas de avaliações no RCO a partir dos valores calculados na planilha.

6.1 Lançar nova avaliação

  1. Clique em Lançar Notas na turma desejada.
  2. Selecione o Trimestre.
  3. Selecione a Avaliação (ATV 1, ATV 2, REC 1…).
  4. Selecione a Data da aula correspondente.
  5. Informe o Valor Máximo da avaliação.
  6. Clique em Buscar Notas da Planilha — a tabela será preenchida.
  7. Revise as notas. Você pode editá-las manualmente antes de enviar.
  8. Clique em Lançar no RCO.
⚠
Alunos inativos são ignorados automaticamente. Células com texto (ex: "AF", "Falta") são tratadas como zero.

6.2 Editar avaliação já lançada

Quando uma avaliação já foi lançada anteriormente no RCO, o software entra em modo de edição automaticamente.

  1. Selecione a avaliação já lançada (aparece com o badge ✓ Lançada).
  2. Clique em Buscar do RCO para carregar as notas atuais do sistema.
  3. Linhas com divergência entre planilha e RCO serão destacadas em amarelo.
  4. Ajuste as notas conforme necessário.
  5. Clique em Editar no RCO.

7. Comentários

Esta aba lança comentários e ocorrências de alunos no RCO, lendo os registros da planilha por período.

  1. Informe o período nos campos De e Até (formato DD/MM/AAAA).
  2. Clique em Buscar comentários.
  3. Os grupos de ocorrências serão listados por data e turma.
  4. Revise os comentários de cada aluno (editáveis antes de lançar).
  5. Clique em Lançar no RCO por grupo, ou Lançar tudo no RCO para todos de uma vez.
ℹ
Grupos já lançados ficam travados (cinza) e não podem ser relançados. O badge ✓ Já lançado confirma o envio.

Se for usar uma planilha própria (via "Usar existente"), ela precisa seguir estas especificações para que o software reconheça os dados corretamente.

9. Fluxos de uso resumidos

Primeiro uso (configuração completa)

  1. Autorizar Google Sheets
  2. Conectar Chrome e fazer login no RCO
  3. Configurar pasta do Google Drive (aba Configurações)
  4. Atualizar Banco no Dashboard
  5. Gerar planilhas na aba Planilhas
  6. Verificar e sincronizar alunos na aba Alunos

Uso semanal típico

  1. Conectar Chrome e confirmar login
  2. Aba Comentários → buscar período da semana → lançar no RCO
  3. Aba Notas → selecionar avaliação → buscar notas → lançar no RCO

Início de trimestre

  1. Atualizar Banco (captura novos alunos e turmas do RCO)
  2. Verificar Todas na aba Alunos → Sincronizar em Lote
  3. Gerar planilhas do novo trimestre (se necessário)

Fim de trimestre

  1. Lançar todas as avaliações e comentários pendentes
  2. Aba Planilhas → Resumo em Lote → selecionar trimestre → gerar resumo
  3. Abrir planilhas no Google Sheets para revisar aprovados/reprovados
